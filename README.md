# 🚗 FiveM Scripts - Simple & Useful

Bienvenue sur mon dépôt GitHub !
Ici, je partage des scripts FiveM simples, utiles et faciles à intégrer pour améliorer vos serveurs.

## 📦 Contenu

Ce repository contient :

Scripts basiques
Systèmes utiles (menus, commandes, etc.)
Ressources optimisées
Code clair et facile à modifier

## ⚙️ Installation

Télécharge le script
Place-le dans ton dossier resources
Ajoute ceci dans ton server.cfg :

```cfg
ensure nom_du_script
```

Redémarre ton serveur

## 🛠️ Technologies utilisées

Lua
FiveM API
Parfois HTML CSS JS pour les interfaces
